# WebShield Frontend

React frontend for WebShield Security Scanner.

## Features
- User dashboard
- Scan initiation & monitoring
- Real-time results display
- AI report viewing
- Responsive design

## Tech Stack
- React + TypeScript
- Tailwind CSS
- React Router
- Axios for API calls
- Vercel deployment
